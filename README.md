# eSBot

**eSBot** is an AI-powered scholarship assistant designed to help Filipino students discover, track, and apply for scholarships efficiently. It receives requests from the eSkolar web app and provides instant, personalized guidance based on student profiles.

## Features

- **Deployment:** Hosted on Railway  
- **Backend:** Built with FastAPI  
- **Database:** ChromaDB for vector storage  
- **AI Model:** LLaMA hosted on Groq for language understanding  
- **Requests Source:** Receiving requests from the eSkolar Web App  
- **Web App Repo:** https://github.com/ChocoCodes/eskolar
